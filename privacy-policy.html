# Politique de Confidentialité - CarWatt **Dernière mise à jour : 20/01/2026** ## Introduction CarWatt ("nous", "notre", "l'application") respecte votre vie privée. Cette politique explique comment vos données sont traitées. ## Collecte de données **CarWatt ne collecte AUCUNE donnée personnelle.** - ✅ Toutes vos données sont stockées localement sur votre appareil - ✅ Aucune connexion à un serveur distant - ✅ Aucun compte utilisateur requis - ✅ Aucun tracking ou analytics - ✅ Aucune publicité ## Données stockées localement L'application stocke uniquement sur votre appareil : - Vos charges de recharge - Vos stations de recharge - Vos trajets - Vos paramètres de configuration Ces données ne quittent jamais votre appareil, sauf si vous les exportez volontairement (fonction Export CSV). ## Permissions requises - **Localisation** : Utilisée uniquement pour afficher votre position sur la carte. Non stockée, non transmise. - **Stockage** : Pour sauvegarder vos données localement sur votre appareil. ## Services tiers ### OpenStreetMap L'application utilise OpenStreetMap pour afficher les cartes. Leur politique de confidentialité : https://wiki.osmfoundation.org/wiki/Privacy_Policy Aucune autre donnée n'est transmise à des tiers. ## Vos droits Vos données vous appartiennent entièrement : - Export à tout moment (CSV) - Suppression complète (via Paramètres) - Aucun compte à supprimer ## Contact Pour toute question concernant cette politique : - Email : [boss@djipi.club] - GitHub : https://github.com/SuperDjipi/CarWatt ## Modifications Cette politique peut être mise à jour. Les modifications seront publiées sur cette page.
